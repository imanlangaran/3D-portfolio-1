# 🚀 3D Portfolio

An interactive 3D developer portfolio built with React and Vite, showcasing projects, skills, and modern frontend animations in a visually engaging experience.

🌐 **Live Demo:** [https://imanlangaran.github.io/3D-portfolio-1/](https://imanlangaran.github.io/3D-portfolio-1/)

---

## ✨ Features

* ⚡ Built with React + Vite
* 🎨 Modern responsive UI
* 🧊 Interactive 3D visuals and animations
* 📱 Mobile-friendly layout
* 🧩 Component-based architecture
* 🚀 Fast development and optimized production builds
* 🛠️ ESLint configuration included
* 🌍 GitHub Pages deployment ready

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

* Node.js
* npm

Check versions:

```bash
node -v
npm -v
```

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/imanlangaran/3D-portfolio-1.git
```

Move into the project directory:

```bash
cd 3D-portfolio-1
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Running the Project

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:5173
```

---

## 🏗️ Production Build

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
